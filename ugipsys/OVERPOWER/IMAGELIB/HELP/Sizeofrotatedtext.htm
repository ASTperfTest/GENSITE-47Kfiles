SIZE OF ROTATED TEXT

When you use Gettextwidth and GetTextHeight it only returns the size of the text as it wasn't rotated (angle 0). But it's not hard to get the size of rotated text. For this, use this formula:

Change the Degrees value to the Angle

radians = (2*PI*Degrees)/360)
cosine = cos(radians)
sine = sin(radians)
'Take 3 corners (not origin) and rotate them
x1 = round(-OldHeight * sine) 'note minus sign
y1 = round(OldHeight * cosine)
x2 = round(OldWidth * cosine - OldHeight * sine)
y2 = round(OldHeight * cosine + OldWidth * sine)
x3 = round(OldWidth * cosine)
y3 = round(OldWidth * sine)

'The rotated corners tell us how big
'to make the new bitmap.
minx = MinIntValue(0,MinIntValue(x1, MinIntValue(x2,x3)))
miny = MinIntValue(0,MinIntValue(y1, MinIntValue(y2,y3)))
maxx = MaxIntValue(x1, MaxIntValue(x2,x3))
maxy = MaxIntValue(y1, MaxIntValue(y2,y3))
'Compute the new bitmap dimensions
NewWidth = maxx - minx
NewHeight = maxy - miny;

function MinIntValue(value1,value2)
if value1 < value2 then
MinIntValue = value1
else
MinIntValue = value2
end function

function MaxIntValue(value1,value2)
if value1 > value2 then
MaxIntValue = value1
else
MaxIntValue = value2
 end function


The newWidth and NewHeight are the width and height of the new image! Good luck
