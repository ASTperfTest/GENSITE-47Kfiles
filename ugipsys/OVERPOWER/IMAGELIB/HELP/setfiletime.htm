SetFileTime

The SetFileTime function sets the date and time that a file was created, last accessed, or last modified.

Function SetFileTime(Filename:String, dCreation: Date, dModify: Date, Access: Date)

Description
This function sets any file date time. The function can be used in the case then creating a picture requires a lot of time.

Example
Set ILIB = server.createobject("Overpower.Imagelib")
cre_date=DateSerial(2001,1,20)+TimeValue("14:15:16")
ILIB.SetFileTime "c:\document.txt",cre_date,cre_date, cre_date
