FindInBinary

The FindInBinary function searches a bytes sequense in binary SAFEARRAY.

Function FindInBinary(searchArray: array of Byte, String: String, From: Integer, Length: Integer, Max: Integer):array of Integer

Description
FindInBinary searches the array SearchArray and look for String. It will start looking for on the position From and ending on length. Max is the max of occurences it will return. The function returns an array. Each element of the array contains the position in the array where the String occurence is found.

Note:   

If you leave Length 0 and from 0, the length will be auto set to high(searcharray) by the function
