SIDEPOLYGON

Draws a polygon with a definied number of sides

Procedure SidePolygon(x: Integer; y: Integer; ray: Integer; sides: Integer; angle: Integer)

Description
Use SidePolygon to draw a polygon with its center on x and y, it's ray (distance from center to one of the borders of a circunference) with a definied number of sides. You can use angle to rotate the Polygon.

Example
Set ILIB = server.createobject("Overpower.Imagelib")
ILIB.WIDTH = 401
ILIB.HEIGHT = 401
ILIB.Polybezier 100,100,20,3, 0 'Draws a triangle
ILIB.PictureBinaryWrite 2,0, ""
