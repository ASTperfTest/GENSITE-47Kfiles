CropFromPicture

Inserts a part from a GIF, JPG or BMP into the current image

Procedure CropFromPicture(Filename: String, posx: integer, posy: integer, x: integer, y: integer, x2: integer, y2: integer)

Description
Use CropFromPicture to insert a part (x, y, x2, y2) from a GIF, JPG or BMP at posx and posy positions on the current image.

Note
Imagelib autodetects the format. First it test the extension, if it's an unknow extension but a supported format it will look the file header and find out which format it is.
