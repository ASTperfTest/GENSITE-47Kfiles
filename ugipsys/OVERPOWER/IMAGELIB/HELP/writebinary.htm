WriteBinary

The WriteBinary writes a array of type SAFEARRAY or part of it to the disk file or to string variable.

Function WriteBinary(safearrey:array of byte, StartIndex: Integer, HowMany: Integer, FileName:String): String

Description
Use WriteBinary for write from the array to a filename on the disk or if filename value is "", it will return an string variable containing all the data. Startindex is the first index of the safearray array to use and HowMany is how many arrays it will use. If you set HowMany and Startindex to 0, Imagelib will auto change HowMany to the number of items on the array.

Example

Set ILIB = server.createobject("Overpower.Imagelib")
DIM data
data = ILIB.ReadBinaryFile "C:\BALL.GIF"
ILIB.WriteBinary data, 0, 0, "C:\COPYOFTHEBALL.GIF"
