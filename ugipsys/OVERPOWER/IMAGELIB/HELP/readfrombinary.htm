ReadFromBinary

Reads an image from an SAFEARRAY and copies it to the current picture.

Procedure ReadFromBinary(Data: array of Byte,Format: Integer)

Description
The ReadFromBinary reads an image from Data and copies it to the current image. You must set the format on the Data.

Format values:

1: Bitmap (.BMP)
2: GIF (.GIF)
3: JPG (.JPG)
4: Windows MetaFile (.WMF)
5: Windows Enhaced Metafile (.EMF)

Notes
Its useful for copying an image from a database for instance.
