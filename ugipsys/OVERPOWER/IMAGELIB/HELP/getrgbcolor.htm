GetRGBColor

The GetRGBRGB function selects a red, green, blue (RGB) color based on the arguments supplied and returns the String (Color).

function GetRGBColor(R: Integer, G: Integer, B: Integer): String (Color)

Description
GetRGBColor returns a color based on the RGB values R, G and B. You can use GetRGBColor with the property FontColor for instance.

Note:    The intensity for each argument is in the range 0 through 255. If all three intensities are zero, the result is black. If all three intensities are 255, the result is white.
