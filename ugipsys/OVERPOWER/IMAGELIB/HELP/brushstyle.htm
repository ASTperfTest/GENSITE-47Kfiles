BRUSHSTYLE

Specifies the pattern for the brush.

property BrushColor: Integer

Description
The Brushstyle property determines the pattern paint by the brush. The possible values of BrushStyle are:

Value Pattern Value Pattern
1 (Solid) 5 (Inclined line)
2 (Clear) 6 (Inclined line)
3 (Horizontal line) 7 (Cross)
4 (Vertical line) 8 (Diagonal cross)
