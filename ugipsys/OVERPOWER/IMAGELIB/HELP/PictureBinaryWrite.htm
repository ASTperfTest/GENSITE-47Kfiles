PICTUREBINARYWRITE

Fills the response.binarywrite with the image and response.contenttype with the right contenttype.

procedure PictureBinaryWrite(format: integer, JPEGQuality: Integer, GIFTransparentColor: String (color))

Description
Use it to fill the response.binarywrite on the ASP file with the desired format and response.contenttype.

The formats values are:

Value: Format:
1: BMP
2: GIF (Uses GIFTransparentColor parameter)
3: JPG (Uses JPGQuality parameter)

Note:    Use only JPGQuality when format parameter is 3 and GIFTransparentColor when format is 2. Otherwise the parameter will be ignored.
