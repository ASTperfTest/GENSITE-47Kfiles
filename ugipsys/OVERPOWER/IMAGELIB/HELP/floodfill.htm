FLOODFILL

Fills an area of the canvas using the current brush.

procedure FloodFill(x: Integer, y: Integer, Color: String (Color))

Description
Use FloodFill to fill a possibly non-rectangular region of the image with the value of BrushColor, BrushStyle. The boundaries of the region to be filled are determined by moving outward from the point (X,Y) until a color boundary involving the Color parameter is encountered.
Floodfill fills all area that has the color indicated by the Color parameter. Stop when another color is encountered.
