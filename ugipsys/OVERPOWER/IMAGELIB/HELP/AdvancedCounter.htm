ADVANCED COUNTER

Here is an example of the counter working:


It's not too hard create a counter using Imagelib. Just follow this steps:

Create 10 images with the same size. Each one contains a number from 0 to 9 (1,2,3,4,5,6,7,8,9,0) and save then to a directory like c:\counter as 1.gif, 2.gif, 3.gif...

Create an ASP file on a directory acessible by the web server and name it counter.asp. And add the ASP tags <% and %>.

Now we want to use a file on the current disk to store the counter values, so add on the ASP file:

contagem = 0
Set Sistema = CreateObject("Scripting.FileSystemObject")
if Sistema.FileExists("c:\counter\counter.val") then
  Set arquivo = Sistema.GetFile("c:\counter\counter.val")
  Set texto = arquivo.OpenAsTextStream(1, -2)
  contagem = texto.readline
  texto.close
end if
contagem = contagem + 1
Set texto = sistema.CreateTextFile("c:\counter\counter.val", true, false)
texto.writeline contagem

We should now create the Overpower Imagelib object by inserting the line:


set ILIB = server.createobject("Overpower.ImageLib")

It's time to get the counter size. To do this, we must know the size of each number image. To do this, insert the follow line:

Dim iwidth, iheight
PictureSize "c:\counter\1.GIF", iwidth,iheight

Now we have to draw the counter image on the picture:


contagemstr = contagem
ILIB.HEIGHT=iheight
ILIB.WIDTH=iwidth*len(contagemstr)+1
for a = 1 to len(contagemstr)
numero = mid(contagemstr,a,1)
ILIB.InsertPicture "c:\counter\"+numero+".GIF",(a-1)*iwidth,0,false,0,0
next

The last step is to make the ASP return the image:


ILIB.PictureBinaryWrite 2, 0, ""

To use the the ASP you just create, add the line to your HTML code:
<IMG SRC="counter.asp">
