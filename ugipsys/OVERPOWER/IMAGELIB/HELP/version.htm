VERSION

Retrieve the current version of Imagelib DLL

Function Version(*major: Integer, *minor:Integer, *fix:Integer): String

Description
Use Version function to retrieve the current ImageLib DLL version. The return value is a string containing the version, ex: "1.512", use the out variables major, minor and fix to retrieve the major (ex: 1), minor (ex: 51) and fix (ex: 2) parts.

Example:   

DIM Major, Minor, Fix
response.write ILIB.Version(major, minor, fix) & "<BR>"
response.write "Major:" & major & "<BR>"
response.write "Minor:" & minor & "<BR>"
response.write "Fix:" & fix & "<BR>"
