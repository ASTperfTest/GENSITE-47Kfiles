ZoomPicture

Zooms the current image

Procedure ZoomPicture(ZoomFactor: integer, ResizeImage: boolean)

Description
ZoomPicture zooms the whole picture (increase the width and height by zoomfactor value) stretching the image. If ResizeImage is true it will resize the image.

Note
ZoomPicture may use antialiase stretch if StretchAntialiased is set to TRUE.
