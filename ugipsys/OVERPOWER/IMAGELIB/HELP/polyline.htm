POLYLINE

Draws a series of lines on the canvas with the current pen, connecting each of the points passed to it in Points.

Procedure Polyline(Points: Array (x,x) of Integer)

Description
Use Polyline to connect a set of points on the canvas. If there are only two points, Polyline draws a single line.

Example
Set ILIB = server.createobject("Overpower.Imagelib")
ILIB.WIDTH = 401
ILIB.HEIGHT = 401
Randomize()
Dim pts(53,1)
for i=0 to ubound(pts)
pts(i,0)=Rnd(1)*401
pts(i,1)=Rnd(1)*401
next
ILIB.PenWidth = 20
ILIB.Polyline pts
ILIB.PictureBinaryWrite 2,0, ""
