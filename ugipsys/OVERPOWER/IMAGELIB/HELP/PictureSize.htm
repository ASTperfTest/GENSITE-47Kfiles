PICTURESIZE

Retrieve the size of a GIF, JPG,BMP image. Now imagelib also support metafiles (.WMF and .EMF).

Procedure PictureSize(Filename: String, *pWidth: Integer, *pHeight:Integer)

Description
Use PictureSize to get the size of an picture filename and retrieve the size of it on the variables pWidth and pHeight.

Note:    Imagelib autodetects the format by extracting the Filename extension. If the extension is not .GIF, .BMP or .JPG it will read the file header to find out the format.
