PenAntialiased

Specifies if Imagelib will use antialiase when drawing lines .

property PenAntialiased: Boolean

Description
Set TRUE to enabled antialiasing for drawing 1 width pens using line function.
Note:    For now, Imagelib only support antialiase for the method line using any PenColor color and it will always draw 1 width lines..

Note:    By default PenAntialiased is set to TRUE.
