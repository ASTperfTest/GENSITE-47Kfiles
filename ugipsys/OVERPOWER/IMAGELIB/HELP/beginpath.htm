BEGINPATH

The BeginPath function opens a path bracket in the image context.

Procedure BeginPath()

Description
After a path bracket is open, an application can begin calling drawing functions to define the points that lie in the path. You can close an open path bracket by calling the EndPath function. When an application calls BeginPath for a device context, any previous paths are discarded from that device context.

Example

<%
Set ILIB = server.createobject("Overpower.Imagelib")
ILIB.WIDTH = 200
ILIB.Height = 60
ILIB.FontFace = "Impact"
ILIB.FontSize = 35
ILIB.BeginPath
ILIB.TextOut "HEY THERE",1,1
ILIB.EndPath
ILIB.PenWidth = 2
ILIB.BrushColor = "#00AAFF"
ILIB.PenColor = "#FFFF00"
ILIB.StrokeAndFillPath
ILIB.PictureBinaryWrite 2,0, "#FFFFFF"
%>
