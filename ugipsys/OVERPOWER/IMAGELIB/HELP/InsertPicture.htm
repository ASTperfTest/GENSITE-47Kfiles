INSERTPICTURE

Inserts a picture from the current server computer into the image. . Now imagelib also support metafiles (.WMF and .EMF).

Procedure InsertPicture(Filename: String, x: Integer, y: Integer, Scale: Boolean, width: integer, height:integer)

Description
Use insertpicture to insert a picture (GIF, BMP or JPG) from the current computer server into the image at position x and y. If Scale is set to true, it will stretch or shrink the image to the new width and height.

Note:    Imagelib autodetects the format by extracting the Filename extension. If the extension is not .GIF, .BMP or .JPG it will read the file header to find out the format.

Note:    Imagelib now supports windows metafile. But it don't use antialised stretch because metafile is vetorial.

Note:    Imagelib by default use antialiasing to create a better effect when scale is set to true. You can set which antialiasing method it will use by change the StretchQuality property (You can disable for faster drawing too).
