CropPicture

Crops the picture using the desired area.

Procedure CropPicture(x: Integer, y: Integer, x2: Integer, y2:Integer)

Description
Use CropPicture to crop the picture starting from x, y and ending on x2, y2. It makes the image smaller.

Example

<%
Set ILIB = server.createobject("Overpower.Imagelib")
ILIB.WIDTH = 240
ILIB.Height = 50
ILIB.FontSize = 30
ILIB.FontColor = "#FF0000"
ILIB.TextOut "HEY THERE",1,1
ILIB.CropPicture 90,1,ILIB.Width,50
ILIB.PictureBinaryWrite 2, 0, ""
%>
