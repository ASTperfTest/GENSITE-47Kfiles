ReadBinaryFile

The ReadBinaryFile function reads the "abstract" binary file into the memory as SAFEARRAY.

Function ReadBinaryFile(Filename: String): array of Byte

Description
The ReadBinaryFile function reads the binary file and returns it in the result of the function

 Example
Set ILIB = server.createobject("Overpower.Imagelib")
DIM data data = ILIB.ReadBinaryFile ("C:\BALL.GIF")
ILIB.WriteBinary data, 0, 0, "C:\COPYOFTHEBALL.GIF"
