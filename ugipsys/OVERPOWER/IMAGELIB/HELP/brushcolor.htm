BRUSHCOLOR

Indicate the color of the brush.

property BrushColor: String (color)

Description
The Brushcolor property determines the color of the brush. This is the color that is used to draw the pattern represented by the style property, not the backcolor of the brush (unless style is bsSolid).
The Brushcolor will be used to draw inside of any shapes like Rectangle, Ellipse, Chord, etc..
Note:    If the value of the Brushstyle property is bsClear, the color of the BrushColor is ignored. Further, any value assigned to BrushColor property is lost when BrushStyle is set to bsClear.
