ROUNDRECTANGLE

Draws a rectangle with rounded corner.

procedure RoundRectangle(x1: Integer, y1: Integer, x2: Integer, y2:Integer, x3: Integer, y3:Integer)

Description
Use Rectangle to draw a rounded rectangle using PenStyle, PenMode, PenColor, PenWidth and fill it with BrushColor, BrushStyle. The rectangle will have edges defined by the points (X1,Y1), (X2,Y1), (X2,Y2), (X1,Y2), but the corners will be shaved to create a rounded appearance. The curve of the rounded corners matches the curvature of an ellipse with width X3 and height Y3.

To draw an ellipse instead, use Ellipse. To draw a true rectangle, use Rectangle.
