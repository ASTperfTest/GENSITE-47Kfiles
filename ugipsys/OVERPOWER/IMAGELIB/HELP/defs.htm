STRING

A string represents a sequence of characters. Strings may range in length from 0 to approximately 2 billion characters.

INTEGER

An integer type represents a subset of the whole numbers. An integer can be from -32,768 to 32,767.

BYTE

Byte is a value that range from 0 to 255.

DATE

VBScript got a loot of methods to deal with dates. To get the current date use now function from VBScript.

BOOLEAN

An boolean type can be TRUE or FALSE.

STRING COLOR
