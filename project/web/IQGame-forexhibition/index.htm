豆仔說：

馬上開始挑戰嚕!! 選擇「開始遊戲」，就可以開始玩遊戲了^^

豆仔說：

開始遊戲後，就可以開始台灣走透透，挑戰您的農業IQ指數嚕^^

每次登入遊戲，會有基本旅行資金1500元，每前進一格，會從資金扣除5元作為旅費。
想要繼續旅行、成功達成各關卡，就要努力地挑戰「農業知識問答」喔!!
每次挑戰問答成功可以獲得獎金$100元，挑戰失敗會扣除$25元，沒有把握的題目可以選擇放棄喔!!

豆仔說：

遊戲進行中，還穿插有機會、命運、和農業相關單位的特殊景點。
挑戰機會成功，可以獲得加倍的$200元，失敗了也不扣分喔!!
走到命運點，將隨機給您好運或壞運，還有農業相關的研究單位資訊讓您更認識台灣的農業單位喔!!

豆仔說：

遊戲每5分鐘會變換季節，讓您一邊玩遊戲、還有我豆仔陪您一起體驗台灣四季的農業之美^^
遊戲中的農業知識問答，也會結合四季和縣市來出題，每個問答都和該作物的季節和產地結合。
