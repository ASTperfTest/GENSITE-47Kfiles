本公司於九十三年九月四日順利完成高企全部資產負債及營業(不含信託業務及其相關資產負債)概括讓與玉山商業銀行新聞稿
內文

九十三年九月四日

高雄區中小企業銀行因調整後淨值已呈負數，經行政院金融重建基金於九十一年二月八日依行政院金融重建基金設置及管理條例第五條及第十條之規定列入處理對象，並委託中央存款保險股份有限公司(以下簡稱存保公司)辦理標售該行事宜。存保公司業於九十三年五月三十一日舉行公開標售完竣，結果由玉山商業銀行得標，行政院金融監督管理委員會並已核准該行全部營業及資產負債(不含信託業務及其相關資產負債)自九十三年九月四日零時起概括讓與玉山商業銀行承受，未來玉山銀行將繼續提供顧客更多元、完整、便利之金融服務。

本次高雄區中小企業銀行債權與債務及營業之讓與承受案，存保公司特此感謝主管機關、行政院金融重建基金之支持、普華財務顧問團隊之協助、各金融同業積極參與競標，尤其是得標機構玉山商業銀行不眠不休、南北奔波之積極妥善規劃交割作業，以及高雄區中小企業銀行全體員工高度配合，致能於短短三個月內順利完成交接及掛牌，不僅創下國內問題金融機構以公開標售方式順利退出市場之首例，亦為金融機構併購案中第一家於第一天開業即完成系統轉換之高效率案例，堪為我國金融改革之優良典範。
