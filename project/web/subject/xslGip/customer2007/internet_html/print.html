蝴蝶蘭的學名是Phalaenopsis sp.，其屬名 Phalaenopsis，是由希臘文的 PHALAENA(n.)（意指彩蛾或蝴蝶）與OPSIS （意指形象）結合而來，喻其花型有如蛾蝶。國人對蝴蝶蘭的型與色皆十分喜愛，稱之為「蘭花之后」。台灣原生的白花蝴蝶蘭(P.amabilis)（俗稱台灣阿嬤），在民國41和42年連續兩年奪得國際蘭展的冠軍獎盃，使得台灣蝴蝶蘭聞名世界，每年產值約有30－40億台幣，日本、美國、中國及歐洲是我們重要的外銷市場，由於各方的努力，目前台灣已成為世界知名的蝴蝶蘭王國。蝴蝶蘭原生的熱帶森林終年高溫，因此溫暖的環境是蝴蝶蘭生長良好的必要條件，其栽培適合的溫度約25－30°C，因品種不同而有差異，18－25°C可促進花梗抽出，溫度太低時不適合花梗生長。蝴蝶蘭開花期通常是在春季，但目前經由栽培技術與溫室環境的控制，幾乎整年都可以有盛開的蝴蝶蘭。
